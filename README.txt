Author: Riccardo Petrini

api - Cog entrypoints and model glue for concatenation-gan inference.
distributions - JavaScript distribution samplers.
finneGAN - Three.js audio-visual latent interface.
gan - Python GAN servers (Flask/FastAPI) and inference helpers.
post-processing - Custom Three.js post-processing passes and variants.
sample - Reference demos, servers, and checkpoints.
shaders - TSL/GLSL noise shader snippets for rendering pipeline.
vector-space - Custom creation of latent values in the latent vector feeding the GANs (heatmap, hover, orbit, slider).
